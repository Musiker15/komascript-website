## What's new?

This release focuses on **stability and maintenance**: several small bugs were fixed,
documentation was clarified, and compatibility notes were added.

## Bug fixes

- Fixed a layout issue with `\KOMAoption{captions=tableheading}` combined with the
  `caption` package
- Better line-breaking in multi-column indexes
- Fixed: incorrect handling of `\thanks` in letters with `fromalign=right`

## Improvements

- Clearer error message for invalid DIV values
- More consistent handling of `\setkomafont{}` and `\addtokomafont{}` when options
  are overridden
- Expanded German manual with new examples for binding correction

## Update

```bash
sudo tlmgr update koma-script    # TeX Live
miktex update                    # MiKTeX
```
