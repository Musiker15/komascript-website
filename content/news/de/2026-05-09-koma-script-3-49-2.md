## Änderung in 3.49.2

Diese Version behebt **genau einen Bug**:

- Eine fälschlicherweise ausgegebene Warnung wurde entfernt.

Das ist der einzige Unterschied gegenüber 3.49.1 — sonst sind keine Änderungen
enthalten.

## Zur Erinnerung: was 3.49.1 brachte

Da der Schritt von 3.49 zu 3.49.1 ähnlich klein war und manche Anwender beide
Releases gleichzeitig einspielen, hier zur Vollständigkeit, was 3.49.1 behoben hatte:

- Eine Fehlermeldung bei Verwendung von `tocbasic` mit anderen Klassen als den
  KOMA-Script-Klassen, wenn das Paket `caption` vor `tocbasic` geladen wurde.
- Ein Fehler bei Verwendung von KOMA-Script-Klassen mit `babel` und einigen
  wenigen Sprachen — z. B. Spanisch.

## Update

```bash
# Vanilla TeX Live
tlmgr update koma-script

# TeX Live über System-Paketmanager (apt, dnf, pacman …)
sudo apt upgrade
sudo dnf upgrade
sudo pacman -Syu

# MiKTeX
miktex update
```

Bei Fragen oder Problemen: [TeX Stack Exchange](https://tex.stackexchange.com/questions/tagged/koma-script)
mit Tag `koma-script`.
