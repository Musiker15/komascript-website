## General

This website processes only the data strictly necessary for operation. There are
**no cookies**, **no tracking**, **no analytics**, and **no external resources** —
fonts, scripts, images, and stylesheets are served exclusively from our own server.

## Hosting

The website is hosted on a server operated in Germany. No personal data is
transferred to third countries.

## Server logs

When visiting this website, technical data is stored in server logs:

- IP address
- date and time of access
- requested URL
- HTTP status code and bytes transferred
- browser identification (User-Agent)
- referrer

This data is processed exclusively for **technical troubleshooting** and **attack
defence** (legal basis: Art. 6 (1) lit. f GDPR — legitimate interest in stable and
secure operation of the website). It is automatically deleted after **at most 14
days**. Log data is not combined with other data sources.

## No cookies, no tracking

This website uses **no cookies** in the legal sense and employs **no tracking
mechanisms**. No data is transmitted to third parties.

The theme preference (light/dark) chosen by the user is stored only in the
browser's `localStorage`. Under § 25 (2) no. 2 TDDDG this storage is strictly
necessary to provide the service expressly requested by the user and contains
**no personal data**. The value never leaves the browser.

## Access protection during the pre-launch phase

During the pre-launch phase the entire website is protected by HTTP Basic
authentication. Submitted credentials (username and password) are checked only
against a server-side password file. They are not stored beyond that; the
username may appear in the server logs described above and is subject to the
retention period stated there. This protection mechanism will be removed at
public launch.

## External links

This website contains links to external services (among others GitHub, Codeberg,
SourceForge, CTAN, Mastodon). When you click such a link, data is transmitted to
the respective provider. Please refer to the privacy notices on those sites.

## Your rights

You have the right at any time to:

- **Access** (Art. 15 GDPR)
- **Rectification** (Art. 16 GDPR)
- **Erasure** (Art. 17 GDPR)
- **Restriction** (Art. 18 GDPR)
- **Data portability** (Art. 20 GDPR)
- **Objection** (Art. 21 GDPR)
- **Complaint** to a supervisory authority (Art. 77 GDPR) — the competent
  authority is the data protection authority of the German federal state in
  which the controller resides.
