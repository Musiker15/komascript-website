The package `gridset` originates from frequent questions in the Usenet group
`de.comp.text.tex`, on the [TeX-D-L](https://www.listserv.dfn.de/sympa/info/tex-d-l)
mailing list, and in the former forum at
[komascript.de](https://komascript.de). The result was a hastily assembled
package that, with the help of a few manually placed markers, enables grid
typesetting.

The package was, however, always intended only as an experimental first
step. Production use was always — and still is — discouraged. Publishing it at
all is now considered a mistake by the author. The only reason it has not been
fully withdrawn is that, against all warnings, a few users have come to rely
on it. Nonetheless, there is no support for this package whatsoever.

Anyone willing to take it over, bring it to actual release readiness, and
maintain it in the future is welcome to get in touch about a maintainer
takeover.

The project is hosted on [GitHub](https://github.com/komascript/gridset). See
the [CTAN catalogue entry](https://www.ctan.org/pkg/gridset) for more.

<Callout type="warning">
This package is looking for a new maintainer. Until one is found there is no
support, and production use is explicitly discouraged.
</Callout>
