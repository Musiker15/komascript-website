The `gridset` package originated from frequent questions in the Usenet group
de.comp.text.tex, as well as on the mailing list
[TeX-D-L](https://www.listserv.dfn.de/sympa/info/tex-d-l) and in the former
forum at [komascript.de](https://komascript.de). Ultimately, this resulted in
a package cobbled together quickly, which, with the help of a few manually
inserted markers, enables typesetting with registers. However, the package was
always intended only as an experimental first step.  Its use in production was
always discouraged and continues to be discouraged. The fact that it was
published at all is now viewed by the author as mistake. The fact that it has
not been completely withdrawn is mainly due to the fact that, despite all
warnings, some users are using it seriously. Nevertheless, there is absolutely
no support for this package. Anyone who would like to take it over, bring it
to actual release readiness, and maintain it in the future is welcome to
contact us to assume maintainer responsibilities.

The project is hosted on [GitHub](https://github.com/komascript/gridset).
Further information can be found in [the CTAN catalog entry](https://www.ctan.org/pkg/gridset).
