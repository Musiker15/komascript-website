Designing title pages is something most authors should not really need to
worry about. Reality is not perfect, however, and so many authors end up
having to.

In 2009 the KOMA-Script author launched “The Title Page Project” as a new area
on [komascript.de](https://komascript.de) to collect real-world title pages
and implement them via a clearly defined interface. In 2021 the project was
moved to GitHub and renamed to `uni-titlepage`. Switching from one title-page
style to another is now very easy. Implementing a new title-page style itself
can still be tricky — but the author of `uni-titlepage` is happy to help if
asked.

The package is hosted on
[Codeberg](https://codeberg.org/komascript/uni-titlepage). See the [CTAN
catalogue entry](https://www.ctan.org/pkg/uni-titlepage) for more.
