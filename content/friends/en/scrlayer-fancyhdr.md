Package `scrlayer-fancyhdr` uses KOMA-Script's `scrlayer` to redefine the page
styles of package [`fancyhdr`](https://www.ctan.org/pkg/fancyhdr). This allows
the combination of features of `fancyhdr` with features of `scrlayer`.

Before KOMA-Script v3.33 `scrlayer-fancyhdr` was part of KOMA-Script
itself. It still depends on KOMA-Script package `scrlayer` at least
version 3.33. It also depends on `fancyhdr`. Currently version 3 and 4 up to
4.0.1 are supported and tested. Basic functionality also works with version 5
and has been tested up to 5.1. But several new features of `fancyhdr`
version 5 are currently not tested with `scrlayer-fancyhdr`.

The package is hosted on
[GitHub](https://github.com/komascript/scrlayer-fancyhdr). Further information
can be found in [the CTAN catalog entry](https://www.ctan.org/pkg/scrlayer-fancyhdr).
