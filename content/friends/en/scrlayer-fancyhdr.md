The package `scrlayer-fancyhdr` uses KOMA-Script's `scrlayer` to redefine the
page styles of the [`fancyhdr`](https://www.ctan.org/pkg/fancyhdr) package.
This makes it possible to combine the features of `fancyhdr` with those of
`scrlayer`.

Before KOMA-Script v3.33, `scrlayer-fancyhdr` was part of KOMA-Script
itself. It still depends on the KOMA-Script package `scrlayer` at least
version 3.33. It also depends on `fancyhdr`. Currently `fancyhdr` versions 3
and 4 up to 4.0.1 are supported and tested. Basic functionality also works
with version 5 and has been tested up to 5.1. Some new features of `fancyhdr`
version 5 have not yet been tested with `scrlayer-fancyhdr`.

The package is hosted on
[GitHub](https://github.com/komascript/scrlayer-fancyhdr). See the
[CTAN catalogue entry](https://www.ctan.org/pkg/scrlayer-fancyhdr) for more.
