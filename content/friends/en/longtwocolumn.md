The package `longtwocolumn` came out of a request. It provides the command
`\longtwocolumn` — a variant of LaTeX's `\twocolumn` that allows the
single-column content of the optional argument to break across multiple pages.
The downside is that footnotes inside the optional argument are not currently
supported.

The package is hosted on
[GitLab](https://gitlab.com/latex9343737/packages/longtwocolumn) under
“[KOMA-Script and Friends](https://gitlab.com/latex9343737)”. See the
[CTAN catalogue entry](https://www.ctan.org/pkg/longtwocolumn) for more.
