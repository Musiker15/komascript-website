Das Paket `contract` ist ein Ableger von KOMA-Script. Solange das Paket Teil
von KOMA-Script war, hieß es `scrjura`. Im Rahmen der Aufteilung von
KOMA-Script in kleinere Projekte wurde es zu einem eigenständigen Projekt. Das
Projekt wird gehostet auf
[Codeberg](https://codeberg.org/komascript/latex-contract). Weitere
Informationen finden Sie im
[CTAN-Katalogeintrag](https://www.ctan.org/pkg/contract).

Siehe auch das experimentelle neue Vertrags-Paket
[`contract-lab`](/de/friends/contract-lab).
